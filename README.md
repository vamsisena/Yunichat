# YuniChat

A real-time chat application with audio/video calling capabilities.

## Quick Start

### Backend
```powershell
cd "YuniChat Backend"
mvn clean install
# Start each service from their respective directories
```

### Frontend
```powershell
cd "YuniChat Tailwind Frontend"
npm install
npm start
```

## Features

-  Real-time chat with WebSocket
-  Audio/Video calls with WebRTC
-  Voice messages
-  Read receipts
-  Typing indicators
-  File sharing
-  User profiles
-  Notifications

## Tech Stack

**Backend:** Spring Boot, PostgreSQL, WebSocket
**Frontend:** React, Tailwind CSS, Redux
**Real-time:** WebRTC, STOMP
